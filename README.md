## Açıklama

Discord.js V14 için Gelişmiş Boş Bot Altyapı

## Kurulum

1-) Konsolda

```bash
npm install
```

Komutunu Çalıştır.

2-) `ayarlar.json` Dosyasını __Eksiksiz__ Doldur.

> Artık Botu Çalıştırmaya Hazırız.

## Çalıştırma

Konsolda

```bash
node bot.js
```

Komutunu Çalıştır.
